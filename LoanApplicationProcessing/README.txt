＊＊日本語＊＊

Loan Application Processing 
  このリポジトリには、ローン申請処理を自動化するための　Loan Application Processing （RPA）ソリューションが含まれています。
  このワークフローは、メール処理、データ検証、Webデータ入力、レポート作成などの繰り返し作業を簡略化し、効率化します。

プロジェクト概要
  このRPAプロセスは、ローン申請処理における以下の主要な手順を自動化します。

メール検索とダウンロード
  当日の日付を含むローンデータのメールを検索します。
  添付ファイルをダウンロードし、次の処理に使用します。

ローンプロセス
  定義済みの条件に基づいてローンデータを評価します。
  評価結果をファイルに入力します。

Webデータ入力
  審査に合格したローンデータをUIBankのWebページに入力します。

審査合格データの反映
  新しいExcelシートを作成します。
  審査に合格したローンデータを入力します。

メール送信
  結果ファイルを関係者にメールで送信します。

プロジェクトファイル
  LoanApplicationProcess RPAシナリオ説明書.pptx、プロジェクトおよびRPAシナリオに関する詳細情報を提供します。
  LoanApplicationProcess RPA実行動画.mp4、AI Indeed RPAツールを使用したRPAプロセスの実行を示すデモ動画です。





＊＊English＊＊

Loan Application Processing 
  This repository contains an automated solution for processing loan applications using Robotic Process Automation (RPA). 
  The workflow simplifies and streamlines repetitive tasks such as email processing, data validation, web data entry, and report generation.

Project Overview
  The RPA process automates the following key steps in loan application processing:

Email Search and Download
  Searches for emails with the current day's date containing loan data.
  Downloads the attached file for further processing.

Loan Process
  Evaluates the loan data based on predefined conditions.
  Updates the file with results of the evaluation process.

Web Data Entry
  Inputs loan data for approved applications into the UIBank webpage.

Approved Data Reflection
  Creates a new Excel sheet.
  Populates it with data for approved loan applications.

Email Sending
  Sends the results file via email to relevant stakeholders.

Project Files
  LoanApplicationProcess RPAシナリオ説明書.pptx: For detailed information about the project and RPA scenario.
