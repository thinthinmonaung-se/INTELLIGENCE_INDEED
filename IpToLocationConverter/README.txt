  取得された情報（国、地域、市）は指定されたExcelファイルの指定列に記録されます。

**English**

IP Address To Location Converter

Project Overview
  The IP Address To Location Converter automates the process of converting IP addresses from an Excel spreadsheet into their respective physical locations, including country, region, and city. The tool uses the website whatismyipaddress.com to retrieve the location data.

Features
  Retrieves physical location details (Country, Region, City) for IP addresses.
  Processes IP addresses directly from an Excel spreadsheet.
  Outputs the retrieved location data back into the same or a new Excel file.
  Ensures 100% accuracy and saves significant manual effort.

How It Works
Data Input:
  The robot reads IP addresses from a specified column in an Excel spreadsheet.
Data Retrieval:
  For each IP address, the robot accesses whatismyipaddress.com.
  It enters the IP address into the search bar and retrieves the corresponding location details.
Data Output:
  The retrieved details (Country, Region, City) are written back into the Excel file, in designated columns.

Requirements
  AI Indeed RPA Tool: For running the automation process.
  Excel: To provide and store IP address data.
  Internet Connection: To access the whatismyipaddress.com website.

Project Files
  IP Address To Location Converter RPAシナリオ説明書.pptx: Detailed documentation of the RPA process.
  IP Address To Location Converter RPA実行動画.mp4: A demonstration video showcasing the RPA process execution using the AI Indeed RPA tool.
