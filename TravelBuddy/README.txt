**日本語**

Travel Buddy RPA（航空便データ収集）
  このプロジェクトは、旅行目的のために航空便データを収集するプロセスを自動化します。

プロジェクト概要
  Travel Buddy RPAロボットは、ユーザーが提供する出発地、目的地、旅行日をもとに航空便情報を収集します。その後、ロボットはフライトを検索し、関連するデータを抽出してExcelファイルに保存します。最後に、そのデータをメールの添付ファイルとしてユーザーに送信します。

動作の流れ
  ユーザー入力: ユーザーが出発地、目的地、旅行日を提供します。
  航空便検索: ロボットが提供された情報を基に利用可能な航空便を検索します。
  データ抽出: 関連する航空便データが抽出されます。
  Excelファイル: 抽出された航空便データがExcelファイルに保存されます。
  メール添付: Excelファイルがメールの添付ファイルとしてユーザーに送信されます。

プロジェクトファイル
  Travel Buddy RPAシナリオ説明書.pptx: プロジェクトの詳細とRPAシナリオの説明を記載した資料です。
  TravelBuddy RPA実行動画.mp4: AI Indeed RPAツールを使用してRPAプロセスが実行される様子を示す動画です。

必要条件
  AI Indeed RPAツール（プロセス実行用）
  メール設定（添付ファイル付きでメールを送信するため）
  Excel（航空便データを保存するため）

使用方法
  このリポジトリをローカルマシンにクローンします。
  Travel Buddy RPAシナリオ説明書.pptxを開き、詳細なRPAプロセスを理解します。
  TravelBuddy RPA実行動画.mp4を視聴して、RPAプロセスが実行される様子を確認します。


**English**

Travel Buddy RPA (Flight Data Collection)
  This project automates the process of collecting air flight data for travel purposes.

Project Overview
  The Travel Buddy RPA robot collects flight information based on user input, including the source, destination, and travel date. The robot then searches for flights, extracts   the relevant data, and saves it in an Excel file. Finally, the bot sends the data as an email attachment to the user.

How It Works
  User Input: The user provides the source, destination, and travel date.
  Flight Search: The robot searches for available flights based on the provided details.
  Data Extraction: The relevant flight data is extracted.
  Excel File: The extracted flight data is saved in an Excel file.
  Email Attachment: The Excel file containing the flight data is sent to the user as an email attachment.

Project Files
  Travel Buddy RPAシナリオ説明書.pptx: Detailed information about the project and RPA scenario.
  TravelBuddy RPA実行動画.mp4: A video showing the execution of the RPA process using the AI Indeed RPA tool.

Requirements
  AI Indeed RPA tool (for running the process)
  Email setup (for sending the email with the attachment)
  Excel (for saving the flight data)

How to Use
  Clone this repository to your local machine.
  Open the Travel Buddy RPAシナリオ説明書.pptx to understand the detailed RPA process.
  Watch the TravelBuddy RPA実行動画.mp4 to observe the RPA process in action.
